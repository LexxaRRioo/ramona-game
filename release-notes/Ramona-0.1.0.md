## Added
- In-app updates via Sparkle. From this version on, Ramona checks for new versions automatically and offers to install them — no more manually checking GitHub. (Note: this release itself still needs a manual download; Sparkle can't update *into* itself.)
- A "Check for Updates…" item in the menu bar.

## Fixed
- Ramona now follows the window she's standing on when it moves up or down, instead of staying at the old height and walking on nothing.
- The release dmg's app bundle is now properly codesigned end to end, avoiding the "damaged" Gatekeeper error.

## Note on installing this version
Since this is the first version with update support, you still need to install it manually: download the dmg below, drag Ramona into Applications, then go through the usual "Apple could not verify" → System Settings → Open Anyway flow once. Every version after this one should offer to update in-app.
